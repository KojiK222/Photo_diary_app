# アプリ名
### Photo diary
# 概要
このアプリでは、登録者は写真や日記のような記録を投稿することができます。また、投稿した写真や日記に対して、コメントをし合ってチャットのような会話をすることができます。登録者のユーザーマイページでは、今まで投稿した写真やプロフィールを見ることができます。
